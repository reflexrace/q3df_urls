Playing online

Create an account: https://www.q3df.org/auth/newaccount This is needed if you would like to log records into the database.

Connecting to a server

View available servers here: https://www.q3df.org/serverlist

Get the IP:PORT of the server

In your console type:

 
xxxxxxxxxx
/connect 5.135.180.180:27961

Connecting using Defrag Launcher

If you installed the defrag url launcher, you can connect by clicking the server ip. The launcher will popup and you will be able to choose your engine and launch the game.

Problems auto-downloading maps

Make sure you have these cvars set:

 
xxxxxxxxxx
/cl_allowdownload 0
/cl_mapAutodownload 1
/dl_source "http://ws.q3df.org/getpk3bymapname.php/%m"

Logging In

Once you are connected to a server you can login:

 
xxxxxxxxxx
/login <username> <password>

Your times will now be saved in the database. Whenever you change servers you'll need to log in again.

Server Commands

Information about server commands can be found here: https://www.q3df.org/wiki?p=16

Some useful commands

 
xxxxxxxxxx
!help                       - list of available commands
!help <cmd>                 - prints description and details of command 
 
xxxxxxxxxx
/callvote kill_OBs          - enable/disable overbounces
/callvote promode           - switch between vq3/cpm physics

Changing Maps

 
xxxxxxxxxx
/callvote map <mapname>     
!recent                     - lists recent maps added to worldspawn
!random                     - lists random maps
!find                       - search for maps

Playing Offline

Downloading Maps

You can download maps here: Worldspawn Archive

Map .pk3 files should be placed inside the defrag subdirectory.

If you know the map name you can use the download command:

 
xxxxxxxxxx
/download <mapname>

Playing Maps

 
xxxxxxxxxx
/map <mapname>
/cpm <mapname>
/vq3 <mapname>
 
xxxxxxxxxx
/devmap <mapname>           - loads map with cheats enabled
/devcpm <mapname>
/devvq3 <mapname>
